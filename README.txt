INTRODUCTION
- This program analyzes an input vCard file to output a list of countries the phone numbers in the vCard file correspond to.
- It also how many of the phone numbers correspond to each country.
- The list is sorted in a descending order of how many phone numbers correspond to each country.
- Finally, the program outputs the total number of phone numbers in the vCard file and the total number of countries they correspond to.

INSTRUCTIONS
- Install python
- Install python packages phonenumbers and pycountry
	On your command line, run
		pip install phonenumbers pycountry
- Run the python script sort.py.
- You will be prompted for a file name. Enter the path of the vCard file here. (If you have the vCard in the same directory as sort.py, you need only enter the file's name.)
